# Boarder 101: Hero Mode

You are the director. Your mission is to go from zero to a living storyboard in minutes.

Open this guide anytime from `Help -> Getting Started (Local 101)`.

## What You Will Learn In This 101 Journey
- Create and open a brand-new project.
- Start drawing immediately on the default starter shot.
- Save confidently and read the title dirty/clean indicator.
- Add a shot at the timeline cursor.
- Insert a new panel inside an existing shot.
- Create and draw on a new layer.
- Create a camera in a shot and confirm it visually.
- Create a new scene attached to your current story flow.
- Save, reopen, and verify your work persists.

## Act 1: Spawn Your Project
1. Launch Boarder.
2. Go to `File -> New`.
3. Fill project name, location, resolution, and canvas preset.
4. Confirm.

Expected result:
- The project opens automatically.
- A default scene/shot/panel is ready.
- You are draw-ready immediately.

## Act 8: Create A New Scene Attached To Current Story Flow
1. Put the cursor on or near the shot where you want to branch/append.
2. Open `Storyboard -> Scene -> New Scene`.
3. In insert mode, choose where to attach it (for example append after current flow).
4. Confirm creation.

Expected result:
- New scene appears with its shots in timeline and shot tree.

## Quick Recovery
`Shot/camera action says no valid selection`
- Move cursor inside a shot and retry.

`Panel or layer not visible`
- Select the shot first, then panel, then layer.

`Changes missing after reopen`
- Save before closing and confirm the title `*` is gone.
